# @milankatira/totion

Your second brain, in the terminal — a fast, keyboard-driven notes TUI with the Snazzy theme.

```sh
npx @milankatira/totion        # run without installing
npm i -g @milankatira/totion   # install globally — gives you the `totion` command
```

This package downloads the prebuilt Go binary for your platform from
[GitHub Releases](https://github.com/milankatira/totion/releases) on first run.

Full documentation, screenshots, and source: **https://github.com/milankatira/totion**
